# Camouflaged-Object-Detection-Using-QFE-with-Q-WaveKAN-Based-Feature-Learning

Camouflaged Object Detection (COD) presents a unique challenge in computer vision where
targets blend seamlessly with backgrounds through color, texture, and edge similarity. QFE-COD
introduces a novel deep learning framework that integrates quantum-classical hybrid processing
with multi-scale frequency domain analysis
 CAM/NonCAM Binary Classifier - PVTv2-B2 backbone achieves 94.23% validation accuracy on
COD10K-v3.
 QFE-COD Segmentation Model-PVTv2-B4 + DWT + FSA + 8-qubit HQCM + FPN-Mamba + Q-
WaveKAN
 SINetV2 Baseline Comparison - Traditional deep learning benchmark for performance
evaluation
 Dataset: COD10K-v3 — 6000 train, 4000 test images
